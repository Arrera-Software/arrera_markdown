# Arrera Markdown

Arrera Markdown is a cross-platform text editor focused on Markdown, designed to integrate seamlessly into the Arrera Software ecosystem.

## Features

*   Full-featured Markdown editor with syntax highlighting.
*   Live preview of the Markdown rendering.
*   Integration with Arrera's AI assistants: Six, Copilote, and Ryley.
*   Direct communication with the main Arrera application.
*   Intuitive and modern user interface.

## Arrera Ecosystem

This project is an integral part of the Arrera software suite. It is designed to work in harmony with other Arrera applications and services, offering a unified and coherent user experience.

## Building from source

This project uses Qt and qmake. To compile the project, follow the steps below:

1.  **Install dependencies:**
    *   Qt 6 
    *   A compatible C++ compiler (GCC, Clang, MSVC)

2.  **Clone the repository:**
    ```bash
    git clone <REPOSITORY_URL>
    cd arrera_markdown
    ```

3.  **Compile the project:**
    ```bash
    qmake
    make
    ```
    On Windows, you may need to use `nmake` instead of `make`.

4.  **Run the application:**
    The executable will be in the build directory.
