# Arrera Markdown

Arrera Markdown est un éditeur de texte multiplateforme axé sur le Markdown, conçu pour s'intégrer de manière transparente dans l'écosystème Arrera Software.

## Caractéristiques

*   Éditeur Markdown complet avec coloration syntaxique.
*   Prévisualisation en direct du rendu Markdown.
*   Intégration avec les assistants IA d'Arrera : Six, Copilote et Ryley.
*   Communication directe avec l'application principale Arrera.
*   Interface utilisateur intuitive et moderne.

## Écosystème Arrera

Ce projet fait partie intégrante de la suite logicielle Arrera. Il est conçu pour fonctionner en harmonie avec les autres applications et services Arrera, offrant une expérience utilisateur unifiée et cohérente.

## Construire à partir des sources

Ce projet utilise Qt et qmake. Pour compiler le projet, suivez les étapes ci-dessous :

1.  **Installez les dépendances :**
    *   Qt 6
    *   Un compilateur C++ compatible (GCC, Clang, MSVC)

2.  **Clonez le dépôt :**
    ```bash
    git clone <URL_DU_DEPOT>
    cd arrera_markdown
    ```

3.  **Compilez le projet :**
    ```bash
    qmake
    make
    ```
    Sur Windows, vous pourriez avoir besoin d'utiliser `nmake` au lieu de `make`.

4.  **Exécutez l'application :**
    L'exécutable se trouvera dans le répertoire de compilation.
